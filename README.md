# Online Shop - Spring Boot Application

## 📌 Features
- JWT Authentication
- REST API

## 🚀 Quick Start
1. Clone the repo
2. Configure `application.properties`
```properties
spring.datasource.url=
```
3. Run:
```bash
mvn spring-boot:run
```

## 🔧 Tech Stack
- Java 21
- Spring Boot 4.0.0
- MySQL
- JWT/OAuth2
