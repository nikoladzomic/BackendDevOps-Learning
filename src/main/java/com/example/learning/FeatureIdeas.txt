PUBLIC:
GET  /api/v1/products              — lista sa filterima i paginacijom ✔
GET  /api/v1/products/{id}         — detalji proizvoda ✔
GET  /api/v1/categories            — sve kategorije ✔

USER:
GET  /api/v1/cart                  — moj košarica ✔
POST /api/v1/cart/items            — dodaj proizvod ✔
PUT  /api/v1/cart/items/{id}       — promeni količinu ✔
DELETE /api/v1/cart/items/{id}     — ukloni iz korpe ✔
POST /api/v1/orders                — napravi order iz korpe ✔
GET  /api/v1/orders                — moje porudžbine ✔
GET  /api/v1/orders/{id}           — detalji porudžbine ✔

ADMIN:
POST   /api/v1/admin/products      — dodaj proizvod ✔
PUT    /api/v1/admin/products/{id} — izmeni proizvod ✔
DELETE /api/v1/admin/products/{id} — ukloni proizvod ✔
PATCH  /api/v1/admin/orders/{id}/status — promeni status ✔
GET    /api/v1/admin/orders        — sve porudžbine ✔
